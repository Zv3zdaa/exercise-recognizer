# Exercise Recognizer & Rep Counter

Система, которая в реальном времени распознаёт физические упражнения и считает повторения.

## Возможности
- Детекция трёх упражнений: приседания, отжимания, планка.
- Подсчёт повторений (для динамических) и секундомер (для планки).
- Визуализация скелета и наложение текста на видео.

## Архитектура
1. MediaPipe Full извлекает 33 ключевые точки тела.
2. Углы локтей и коленей рассчитываются и сглаживаются медианным фильтром.
3. Классификация по скользящему окну (15 кадров = 0.6 с).
4. RandomForest (или LightGBM) обучен на статистиках окна.
5. Конечный автомат подсчитывает повторения; гистерезис предотвращает ложные переключения.

## Точность
- RandomForest: 96% (f1-score macro).
- LightGBM: 94%.

## Структура репозитория
- `utils.py` – функции извлечения углов, признаков, отрисовки.
- `train.py` – сбор данных, обучение модели, сохранение.
- `inference.py` – обработка видео в реальном времени.
- `data/` – папки с видео для каждого упражнения.
- `exercise_model.pkl` – сохранённая модель и список признаков.
